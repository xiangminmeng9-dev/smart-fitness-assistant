# Smart Fitness Assistant

智能健身助手Web应用

## 项目概述

基于用户身体数据和健身目标，AI生成个性化健身计划的Web应用。

## 技术栈

- 前端: TypeScript + React
- 后端: Python + FastAPI/Flask
- 数据库: MySQL
- AI: Claude Opus 4.6

## 功能特性

1. 用户注册/登录
2. 身体数据填写
3. AI生成个性化健身计划
4. 每日健身计划展示与完成跟踪
5. 天气显示和鼓励语
6. 健身小贴士
7. 用户信息管理

## 项目结构

```
smart-fitness-assistant/
├── frontend/          # 前端TypeScript代码
├── backend/           # 后端Python代码
├── database/          # 数据库脚本
├── docs/              # 文档
└── scripts/           # 部署脚本
```
